# Directory Traversal Attack using Burp Suite

This project demonstrates a Directory Traversal vulnerability using
Burp Suite and concepts from PortSwigger labs.

## Project Structure
- `src/` – Vulnerable demo application
- `documentation/` – Report, attack steps, prevention, screenshots

## Tools Used
- Burp Suite
- Python (Flask)
- PortSwigger Labs

⚠️ All screenshots were captured in a controlled lab environment
(Metasploitable2) for academic and educational purposes only.
