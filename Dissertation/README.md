Purpose

This repository contains the software artefact developed for a Master’s dissertation on privacy-preserving behavioural analysis of Monero transactions using unsupervised machine learning.

System Requirements

Operating System: Windows 11

Python: 3.10 or higher

Disk space: ~10–15 GB (for pruned Monero blockchain + data)

Internet connection (for blockchain sync or explorer fallback)

Software Dependencies

All Python dependencies are listed in requirements.txt.

Project Setup
1. Extract project
cd C:\
cd monero-pp-analysis

2. Create virtual environment
python -m venv .venv
.venv\Scripts\activate

3. Install dependencies
pip install -r requirements.txt

Dataset Preparation (Offline Mode)

The offline dataset consists of ~30,000 Monero transactions extracted from a fixed block range.

Required file:

data/processed/features_with_scores.csv


This file is generated by running:

python src/data_extraction.py
python src/feature_engineering.py
python src/ml_module.py


(Pre-generated datasets can be used directly if provided.)

Optional: Monero Node (Live Mode)

To enable live transaction lookup via RPC:

Download Monero binaries from:

https://www.getmonero.org/downloads/


Run pruned node:

monerod.exe --prune-blockchain --data-dir C:\MoneroData


Live analysis is optional; the system works fully in offline mode.

Running the Application

Start the Streamlit dashboard:

streamlit run app/streamlit_app.py


Open browser at:

http://localhost:8501
